# Self Love Animation 

A visually soothing, interactive animation that promotes self-love and positivity.
The project displays your name with a typing effect, timed motivational quotes, and a glowing smiley, 
all set against a dynamic particle background.

---

##  Features

- **Typing Effect for Name:**  name is animated letter by letter.
- **Timed Motivational Quotes:** Quotes appear sequentially to encourage self-love and mindfulness.
- **Glowing Smiley Animation:** A glowing smiley appears to enhance the visual experience.
- **Responsive Design:** Works on all screen sizes with a full-screen immersive experience.

---

##  Technologies Used

- **HTML5** – Structure of the webpage  
- **CSS3** – Styling, animations, and transitions  
- **JavaScript (ES6)** – Typing effect, timed messages, and particle animation  
- **Canvas API** – For particle background rendering
